# Energiklok

Smartare energibeslut för BRF:er och villaägare i Sverige.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Next.js 16** (App Router)
- **Tailwind CSS 4**
- **TypeScript**
- **Recharts** for calculator visualizations
- **Lucide React** for icons
